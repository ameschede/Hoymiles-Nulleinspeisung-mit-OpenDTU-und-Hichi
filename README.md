# Nulleinspeisung für Hoymiles-Wechselrichter mit OpenDTU & Python Steuerung

Dies ist ein Python-Skript, das den aktuellen Hausverbrauch aus einem Tasmota-basierten Lesekopf (z.B. Hichi IR) ausliest, die Nulleinspeisung berechnet und die Ausgangsleistung eines Hoymiles-Wechselrichters mit Hilfe von OpenDTU entsprechend anpasst. Somit wird kein unnötiger Strom ins öffentliche Netz abgegeben und ein vorhandener Akku bestmöglich genutzt.

Dieses Skript ist ein Fork von: https://github.com/Selbstbau-PV/Selbstbau-PV-Hoymiles-nulleinspeisung-mit-OpenDTU-und-Shelly3EM
